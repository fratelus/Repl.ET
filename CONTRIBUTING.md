# Contributing to Repl.ET

Thank you for your interest in contributing to Repl.ET! 🎉

## 🐛 Reporting Issues

- Use [GitHub Issues](https://github.com/fratelus/Repl.ET/issues) to report bugs
- Provide clear reproduction steps and expected vs actual behavior
- Include relevant system information (OS, Python version, etc.)

## 💡 Suggesting Features

- Open a [GitHub Issue](https://github.com/fratelus/Repl.ET/issues) with the "enhancement" label for feature requests
- Explain the use case and how it would benefit the eye-tracking research community

## 🔧 Development

1. **Fork and Clone**:
   ```bash
   git clone https://github.com/your-username/Repl.ET.git
   cd Repl.ET
   ```

2. **Install Dependencies**:
   ```bash
   pip install -r config/requirements.txt
   ```

3. **Run Tests**:
   ```bash
   pytest tests/ -v
   ```

4. **Make Changes**:
   - Follow existing code style
   - Update documentation as needed
   - Add tests for new functionality

5. **Submit Pull Request**:
   - Create a descriptive PR title and description
   - Reference any related issues
   - Ensure all tests pass

## 📋 Guidelines

- **Documentation**: Update README.md and docs/ for user-facing changes
- **Testing**: Maintain 100% test coverage
- **Schemas**: Validate any JSON schema changes with existing data
- **Reproducibility**: Ensure changes don't break the scoring system

## 🤝 Code of Conduct

Be respectful, inclusive, and collaborative. This project serves the academic research community.
