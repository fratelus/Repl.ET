# Templates Directory

This directory contains template structures and files for the Repl.ET framework.

## Contents

- **`data_structure/`**: Template directory structure for organizing eye-tracking study data
  - Contains empty directories with `.gitkeep` files to preserve the recommended folder structure
  - Copy these directories to your project root when setting up a new study

## Usage

When starting a new eye-tracking study with Repl.ET:

1. Copy the contents of `data_structure/` to your project root
2. Each directory represents a different aspect of your study data
3. Follow the schema definitions in the `schemas/` directory for data format requirements

This keeps the root directory clean while providing users with the recommended structure when needed.
